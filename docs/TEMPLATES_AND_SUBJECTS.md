### Key Principles for AI Content Generation

1. **Grade-Level Language**
   - K-2: Simple sentences, basic vocabulary
   - 3-5: Moderate complexity, some academic vocabulary
   - 6-8: More complex, subject-specific terminology

2. **Content Accuracy**
   - All facts must be educationally accurate
   - Age-appropriate explanations
   - Align with common curriculum standards

3. **Engagement**
   - Use interesting examples
   - Connect to real-world applications
   - Avoid dry, textbook-style language

4. **Flexibility**
   - Generate more content than needed
   - Allow teacher customization
   - Provide answer keys automatically

### Sample Prompt Structure

```
You are creating a {worksheet_type} for {grade_level} students about {topic}.

Generate:
- {specific_content_requirements}
- Use vocabulary appropriate for {grade_level}
- Include {number} of {items}
- Ensure educational accuracy
- Make it engaging for young learners

Return in JSON format with structure:
{expected_json_structure}
```
