### 3.2 Extraction Plan

PowerPoint files (.pptx) are actually ZIP archives containing images. To extract:

1. **Rename** `.pptx` to `.zip`
2. **Unzip** the file
3. **Navigate** to `ppt/media/` folder
4. **Copy** all image files (usually .png, .emf, .wmf)
5. **Convert** EMF/WMF to PNG if needed
6. **Rename** with descriptive keywords
